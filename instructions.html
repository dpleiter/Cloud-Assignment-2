How it works..

Firstly, you must ensure that you have a registered account, if you don't, you must register with a valid email.
After registering successfuly, you can then login. You are then taken to the main page, which should display a bar where you can enter any address or place.
This will then take you to another page which will display a map with the address input in the page before. It will also display details of similar nearby organisations/places.
You can then save it for the next time you log in, toggle the markers on the map, or sign out.
